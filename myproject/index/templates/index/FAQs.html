  • What technology do we implement in our website?

    • 1. Django(https://www.djangoproject.com/)

    Use the python framework django to build the main components of the application. These components will include:(1) a main front end, where the public will have access to the main resources and database; (2) Interactive phylogenetic tree, where users are can interact with the cataloged bacterium; (3) a main administrator back end to update components in the front end.

    2. D3.js ​ (http://d3js.org/)

    The Javascript framework that will be used to create interactive charting diagrams.

    3. BLAST ​ (http://www.ncbi.nlm.nih.gov/books/NBK279690/)

    A command tool developed in by the NCBI that will be utilized when comparing or searching for a sequence of nucleotide base pairs or amino acids against the database of human microbiome DNA.

    4. PHYLIP​ (http://evolution.genetics.washington.edu/phylip.html)

    A free package of programs to infer phylogenies with the help of genomic sequences.

    5. CLUSTALW2​ (http://helixweb.nih.gov/multi­align/man/clustalw.1.html)

    To carry out multiple sequence alignment of three or more biological sequences,generally protein, DNA, or RNA.

    6. MySQL ​ (https://www.mysql.com/)

    Use a database relational storage to hold relevant/persistent information for the web application and genome sequences.

    7. WSGI ​ (http://wsgi.readthedocs.org/en/latest/index.html)

    A python oriented ​ Web Server Gateway Interface.

    8. Apache ​ (http://httpd.apache.org/)

    A HTTP server used alongside WSGI through the module mod_wsgi that is provided by Apache.
